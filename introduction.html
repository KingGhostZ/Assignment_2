Recent Projects

Automatic Nightlight Design

This nightlight is designed with only analog hardware to shut off the Transistor if light is coming through. However,
if there is no light, then the nightlight should turn on. We utilized a photoresistor in parallel with the transistor where the base is connected.
That means with no light, there is a lot of resistance, meaning current will turn on the transistor, however, with light there is
no resistance in the transistor. Which means current will flow freely through it and never touch the transistor.

Bluetooth Connection with Microcontroller
